ViterbiJob class
================

Typically, methods of the :class:`~gwlab_viterbi_python.gwlab_viterbi.GWLabViterbi` class will return instances of the ViterbiJob class.
Instances of this class are representations of the the Viterbi jobs that are stored on the GWLab server.
